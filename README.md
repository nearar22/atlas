# Atlas: A Gazetteer of the Shared World

An on-chain AI canon-keeper for a single shared fictional world drawn on a 12x12 grid (A1 through L12). An explorer claims a region coordinate and writes its lore; an injection-resistant AI cartographer rules the claim CANON, CONTESTED, or APOCRYPHA under GenLayer validator consensus.
